# 🚀 Easy Guide to Upload Your Files to GitHub (No Coding Required!)

## The Easiest Way: GitHub Desktop

### Step 1: Download GitHub Desktop
1. Go to: **https://desktop.github.com/**
2. Click the big **"Download for Windows"** button
3. Run the installer (just click "Next" through everything)

### Step 2: Sign In to GitHub
1. Open GitHub Desktop
2. Click **"Sign in to GitHub.com"**
3. Enter your GitHub username and password
4. Click **"Authorize desktop"**

### Step 3: Add Your Project
1. In GitHub Desktop, click **"File"** → **"Add local repository"**
2. Click **"Choose..."** button
3. Navigate to: `C:\Users\brayd\Desktop\ADVERSIQ-Intelligence\ADVERSIQ-Intelligence-main`
4. Click **"Select Folder"**

### Step 4: Connect to Your GitHub Repository
If it says "This directory does not appear to be a Git repository":
1. Click **"create a repository"** link
2. Then click **"Publish repository"**
3. **IMPORTANT**: Uncheck "Keep this code private" if you want it public
4. In the repository URL, paste: `elizabethlwalls1972-crypto/ADVERSIQ-Intelligence`
5. Click **"Publish repository"**

### Step 5: Upload Your Changes
1. You'll see a list of changed files on the left
2. At the bottom left, there's a box that says "Summary (required)"
3. Type: `Updated landing page with new orange design`
4. Click the blue **"Commit to main"** button
5. Click **"Push origin"** at the top

### ✅ Done! 
Your files are now on GitHub at:
**https://github.com/elizabethlwalls1972-crypto/ADVERSIQ-Intelligence**
