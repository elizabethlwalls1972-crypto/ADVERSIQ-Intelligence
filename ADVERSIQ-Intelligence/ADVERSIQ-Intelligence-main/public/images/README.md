## Design Guidelines

### Color Palette
- Primary: Orange (#E67E22)
- Dark: Navy/Dark Blue (#1a1a2e)
- Accent: Light Blue (#00d4ff)
- Background: Dark gradients with subtle patterns

### Style Requirements
- **Professional & Modern**: Clean lines, minimal clutter
- **High Resolution**: At least 2x resolution for retina displays
- **Consistent Style**: All images should share similar visual language
- **Cybersecurity Theme**: Incorporate network patterns, data flows, shields
- **Orange Accents**: Use ADVERSIQ brand color throughout

### Inspiration
- Reference the CyberGuard template style
- Professional business photography for hero section
- Technical diagrams with modern UI elements
- Isometric or 3D illustrations for features

## Image Formats
- **PNG**: Preferred for illustrations with transparency
- **JPG**: Acceptable for photographs
- **SVG**: Ideal for icons (can be converted to PNG)

## Fallback Behavior
If images are not found, the page will display gradient backgrounds as placeholders. However, for the best user experience, all images should be provided.

## Testing
After adding images, test the page at:
- http://localhost:3001

Verify that:
- All images load correctly
- Images are sharp on high-DPI displays
- Page loads quickly (optimize images if needed)
- Images look good on mobile devices
